제목 요소

HTML에는 h1 부터 h6 까지 총 6개의 제목 tag가 존재한다.
이들은 단순히 크기의 구분만을 하는 것이 아닌, 해당 페이지에 포함된 내용의 중요도를 나타내는 기능을 한다.
아래는 h1 ~ h6 태그를 사용하면 표현되는 모습이다.

h1. 가장 중요한 제목

h2. 두번째로 중요한 제목

h3. 세번쨰로 중요한 제목

h4. 네번쨰로 중요한 제목

h5. 다섯번째로 중요한 제목
h6. 제목들 중 가장 낮은 중요도의 제목

+ h1 tag는 하나의 페이지에 한번만 사용하는 것이 일반적이다. 보통 웹사이트의 로고 등에 사용한다.

내용 강조 표현

HTML에는 내용을 강조하는 기본 코드들이 존재한다.
strong. 임팩트의 기능을 가진 굵은 글씨
em. 권유 형식의 기능을 가진 기울임 글씨
ins. 텍스트의 아래에 밑줄을 포함한 글씨
del. 텍스트의 중간에 삭제를 의미하는 줄이 포함된 글씨

문서의 구역을 분리하는 요소

HTML에서 문서의 내용을 작성하는 구간을 분리하는 tag를 의미한다.
div, p, pre 등이 있다.
div. 단순히 내용의 구역을 분리하는 용도로 사용한다. CSS tag를 통해 스타일을 부여해주면 다른 구역과의 분리를 더욱 명확히 할 수 있다.
p. 문서의 내용을 작성하는 구간으로 사용한다. p 태그의 안에서 뛰어쓰기를 하더라도 실제 웹 페이지에는 줄바꿈이 적용되지 않는다. 줄바꿈을 할 때에는 필히 br. 태그를 작성한다.
pre. p 태그와 비슷하나, br. 코드를 사용하지 않아도 원하는 대로 줄바꿈을 할 수 있다. 단, 활용도가 그리 높지 않아 실무에서는 코드의 삽입 기능 정도로만 사용한다.
